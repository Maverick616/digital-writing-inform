A work of interactive fiction.

HoloInform was created with Inform and has IFID 87144C54-D496-4DC0-8F75-A6635B9ABBD1. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for MacOS, Linux or Windows; Spatterlight for MacOS; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
